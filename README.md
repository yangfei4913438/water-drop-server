## Description

一个基于`graphql`的`API`服务, 底层使用`nestjs`, 短信服务和图片存储等，则是使用第三方阿里云全家桶处理。

- PC客户端: [water-drop-pc](https://github.com/yangfei4913438/water-drop-pc) 

- H5客户端: [water-drop-mobile](https://github.com/yangfei4913438/water-drop-mobile)


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
